Веб-приложение для создания опросов, сбора ответов и анализа результатов. Сервис разработан как дипломная работа и позволяет развернуть полнофункциональную систему на собственном сервере, без привязки к облачным платформам и их ограничениям.

## 📖 О проекте

Этот сервис предназначен для организаций, учебных заведений и любых групп пользователей, которым требуется гибкий инструмент для проведения опросов. Система построена на архитектуре "клиент-сервер" с использованием REST API, что обеспечивает её независимость и простоту масштабирования.

**Зачем нужен этот сервис, если есть Google Forms?**
Готовые решения часто имеют лимиты на количество ответов, собирают данные на чужих серверах, могут быть платными и не всегда соответствуют внутренним политикам безопасности. Наш сервис вы полностью контролируете, разворачивая на своей инфраструктуре.

## ✨ Основные функции

*   **Управление опросами:** Создание, редактирование и удаление опросов.
*   **Гибкие настройки вопросов:** Поддержка вопросов с выбором одного варианта, нескольких вариантов и текстовым ответом.
*   **Настройка параметров опроса:** Возможность установить анонимность, срок действия опроса и ограничение на одно прохождение с одного устройства/пользователя.
*   **Публикация и доступ:** Генерация уникальной ссылки для прохождения опроса.
*   **Прохождение опроса:** Респонденты отвечают на вопросы по ссылке без необходимости регистрации, с любого устройства (компьютер, планшет, смартфон).
*   **Анализ результатов:** Просмотр сводных результатов по каждому опросу, экспорт данных в формат CSV.
*   **Ролевая модель:**
    *   **Создатель опроса:** Может создавать, публиковать свои опросы и просматривать результаты.
    *   **Респондент:** Проходит опрос по публичной ссылке.
    *   **Администратор:** Управляет пользователями и правами доступа.

## 🛠️ Технологический стек

*   **Серверная часть (Backend):**
    *   Язык: **Java 25**
    *   Фреймворк: **Spring Boot 4.0.2**
    *   Безопасность: **Spring Security**
    *   Доступ к данным: **Spring Data JPA**
*   **Клиентская часть (Frontend):**
    *   **HTML5, CSS3, JavaScript** (чистый JS, без фреймворков)
*   **База данных:**
    *   **PostgreSQL 12+**
*   **Сборка и развертывание:**
    *   **Maven**
    *   Веб-сервер (опционально): **Apache** или **Nginx** (для проксирования)

## 💻 Требования к системе

### Для сервера
*   Процессор: 2 ядра, от 2 ГГц
*   ОЗУ: от 4 ГБ
*   Диск: от 20 ГБ свободного пространства
*   ОС: **Linux** (рекомендуется) или **Windows Server**
*   Программное обеспечение:
    *   **JDK 25**
    *   **PostgreSQL 12+**
    *   **Maven 3.8+**
    *   Опционально: веб-сервер для обратного прокси (Apache/Nginx)

### Для клиента (пользователя)
*   Любое устройство с современным браузером (**Chrome**, Firefox, Safari, Edge)
*   Подключение к интернету или локальной сети, где развернут сервис
